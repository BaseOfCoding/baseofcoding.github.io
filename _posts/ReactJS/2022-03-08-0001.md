# 설치방법

## - 터미널 또는 CMD 창에서 명령어 입력하기

react.js를 설치할 수 있는 환경이 갖춰졌다는 가정하에 작성이 되었습니다.

```bash
npx create-react-app '프로젝트 이름' --template typescript
```

이렇게 입력을 하게 되면 React.js 환경에서 Typescript를 사용할 수 있게 됩니다.
